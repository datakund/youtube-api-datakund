Click View Replies
**************************************************
It will click on view replies button of the comment whose text is passed in ``comment``.

**body**: returns data

**success_score**: api success rate

**errors**: errors encountered in api 

Here is the code:-

   :return: {"body": [{}], "success_score": "100", "errors": []}
